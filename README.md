# Projeto28

Será que é possivel gerar música usando treinamento de LLM?

Eu pelo menos não consegui após dias tentando...

A LLM alucinou e só gerava 3 ou 4 notas por arquivo.

A solução foi usar Transformers, treinando com 100 músicas de piano no formato Midi.

O objetivo do projeto é aprender mais sobre Transformers e entender como aprende sequências.

Use este [site](https://pianotify.com/import-midi-file) para fazer upload do .mid e ouvir.
